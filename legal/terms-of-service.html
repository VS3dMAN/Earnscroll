2. Description of Service

EarnScroll is a screen-time management application that gamifies physical activity. Users earn minutes of app usage time by completing real exercises (squats, pushups, planks) detected through the device's camera using on-device machine learning. When earned time is depleted, the App can block access to user-selected applications using Android's accessibility service.

Key features include:

  • Exercise detection via on-device pose estimation (camera-based)
  • A time bank that tracks earned screen-time minutes
  • App blocking for user-selected applications when time runs out
  • Workout history and streak tracking
  • Emergency pause functionality (up to 3 per day, each granting 5 minutes of free screen time)
  • Free and Pro tiers with varying feature access
